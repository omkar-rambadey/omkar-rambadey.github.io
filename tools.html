Interactive Research Tools

A collection of open-source utilities for X-ray scattering and data visualization. These tools run locally in the browser using Python (PyScript).

1. Ewald Sphere Calculator

This tool calculates the radius of the Ewald sphere based on the incident X-ray wavelength \( \lambda \). In reciprocal space, the radius \( R \) is defined as:

\[ R = \frac{1}{\lambda} \]
Enter a value and click calculate...

2. Future Python GUIs

I am currently porting my desktop GUIs (Tkinter/PyQt) to the web. Upcoming features include:

  • Reciprocal Space Mapper: Visualize \( (hkl) \) planes relative to the Ewald sphere.
  • Unit Cell Converter: Transition between real space lattice parameters and reciprocal lattice constants.
